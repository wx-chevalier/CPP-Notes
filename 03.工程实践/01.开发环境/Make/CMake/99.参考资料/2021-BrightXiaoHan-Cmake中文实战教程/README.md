# Cmake 中文实战教程

本项目以代码讲用法，旨在帮助初学者学习 CMake 的基本用法，也会结合实际开发经验将最核心最常用的用法使用实际的代码案例进行讲解。该项目很难涵盖 CMake 的所有用法，本项目只会扮演一个引路者的角色，在实际的项目开发中，要学会根据需求灵活查阅官方文档解决问题。(本教程的所有章节使用 c++以及 linux 环境演示)
