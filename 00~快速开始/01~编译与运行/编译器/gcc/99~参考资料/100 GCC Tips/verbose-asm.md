# 生成有详细信息的汇编文件

## 例子

```cpp
    #include <stdio.h>

    int main(void)
    {
      int i;

      for (i = 0; i < 10; i++)
        printf("%d ", i);
      putchar ('\n');

      return 0;
    }
```

## 技巧
