# 把选项传给连接器

## 例子

    #include <stdio.h>

    int main (void)
    {
      puts ("Hello world!");
      return 0;
    }

## 技巧

使用`-Wl,option`可以将选项`option`传递给连接器。

注意，逗号和选项之间不能有空格。一种常见用法，就是让连接器生成内存映射文件，例如：
